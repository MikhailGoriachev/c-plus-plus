Dynamic8. 
Даны два непустых стека целых чисел. Переместить все элементы из первого стека во второй 
(в результате элементы первого стека будут располагаться во втором стеке в 
порядке, обратном исходному).

Stack stack1(arr1, n1);
Stack stack2(arr2, n2);

while (!stack1.isEmpty()) {
    v = stack1.pop();
	stack2.push(v);
	
	// stack2.push(stack1.pop());
}

Dynamic 9.
Даны два непустых стека целых чисел. Перемещать элементы из первого стека во второй, 
пока значение вершины первого стека не станет четным (перемещенные элементы первого стека
будут располагаться во втором стеке в порядке, обратном исходному). Если в первом стеке 
нет элементов с четными значениями, то переместить из первого стека во второй все элементы.

Stack stack1(arr1, n1);
Stack stack2(arr2, n2);

while(!stack1.isEmpty() && (stack1.peek() & 1) != 0) {
    v = stack1.pop();
	stack2.push(v);
}
  

Dynamic10.
Дан непустой стек целых чисел. Создать два новых стека, переместив в первый из них все 
элементы исходного стека с четными значениями, а во второй — с нечетными (элементы в 
новых стеках будут располагаться в порядке, обратном исходному; один из этих
стеков может оказаться пустым).
Stack stack1(arr1, n1);
Stack stack2, stack3;

while(!stack1.isEmpty()) {
    v = stack1.pop();
	if ((v & 1) == 0)
        stack2.push(v);
    else
       stack3.push(v);
}
