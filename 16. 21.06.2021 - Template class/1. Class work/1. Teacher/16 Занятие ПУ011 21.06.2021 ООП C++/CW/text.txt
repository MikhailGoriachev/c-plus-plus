Статичекий полиморфизм

Ограничения на использование шаблонов.

Шаблоны                                          void *
Класс по умолчанию генерирует 4 метода:
    конструктор по умолчанию,
    конструктор копирования,
	оператор присваивания
    деструктор.
Данные методы не могут быть шаблонными.
Также виртуальный метод не может быть одновременно и шаблонным.

Шаблонные функции не могут иметь шаблонные параметры
по  умолчанию.
