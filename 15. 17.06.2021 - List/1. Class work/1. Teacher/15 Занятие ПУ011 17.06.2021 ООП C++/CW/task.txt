Разработать приложение с использованием меню для закрепления работы 
со списком – доработать класс List – односвязный список целых чисел: 
реализовать следующие операции со списком. 
В приложении продемонстрировать разработанные операции:
    • get(postion)         – возвращает значение элемента списка 
	                         в заданной позиции
    • put(position, value) – записывает значение value в заданную 
	                         позицию списка
    • операция []          - для доступа к элементам списка по заданному 
	                         значению позиции position
    • addAll(data[], n)    - добавление массива в конец списка
    • insertAll(position, data[], n) - вставка массива data[] перед элементом
                          	 списка с индексом position
